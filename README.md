## S3

### Introduction

Dans le cadre de la SAE S3 nous devons créez un jeu Pipopipette jouable à 2 joueurs en Python à l'aide de la bibliothèque PyGame.

### Ce que permet le jeu

Le jeu devras être jouable en 1 VS 1 en implémentant un système de compte et de sauvegarde du nombre de partie gagner

Le map devras être modulable (grande, moyenne, petite, personnalisé)
